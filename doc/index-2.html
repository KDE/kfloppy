2. Installation

2.1 How to obtain KFloppy Formatter

KFloppy Formatter is part of the KDE project http://www.kde.org. KFloppy Formatter can be found in the kdeutils package on ftp://ftp.kde.org/pub/kde/, the main ftp site of the KDE project.

2.2 Requirements

In order to successfully compile KFloppy Formatter, you need the latest versions of the KDE libraries. All required libraries as well as KFloppy Formatter itself can be found on ftp://ftp.kde.org/pub/kde/.

2.3 Compilation and installation

In order to compile and install KFloppy Formatter on your system, type the following in the base directory of the KFloppy distribution: 

% ./configure
% make
% make install

Since KFloppy Formatter uses autoconf you should have not trouble compiling it. Should you run into problems please report them to the KDE mailing lists.
